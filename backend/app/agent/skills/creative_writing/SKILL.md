# 知乎创作辅助

## 工作流程

### 第一步：理解创作意图
确认用户想创作的话题、内容形式（知乎回答/想法/文章）、风格偏好。

### 第二步：素材收集
1. 调用 `zhihu_search` 搜索知乎站内相关高质量讨论
2. 如需要，调用 `zhihu_global_search` 获取站外参考
3. 如需要，调用 `zhihu_direct_answer` 获取专业回答作参考

### 第三步：分析与构思
1. 提取已有讨论的主流观点和争议点
2. 发现尚未被充分讨论的独特角度
3. 准备可引用的数据、案例、专家观点

### 第四步：生成内容
根据用户选择的方向，提供内容大纲或精炼的想法内容。

### 第五步：润色优化（可选）
用户提供草稿后优化标题、结构、论据和语言。

## 知乎内容创作原则
1. **信息密度高**：知乎用户期望看到有深度的内容
2. **逻辑清晰**：论点-论据-结论的清晰结构
3. **有独特视角**：避免人云亦云，提供新角度
4. **数据说话**：尽量用数据和案例支撑观点
