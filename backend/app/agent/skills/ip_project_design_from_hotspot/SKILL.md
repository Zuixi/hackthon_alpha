## 工作流程

### 第一步：热点价值评估
分析该热点是否适合开发成长篇内容（是否有足够延展性、是否有爽点支撑、是否符合用户创作风格）

### 第二步：方案头脑风暴
提供3-4个不同角度的创作方案，每个方案包含：核心设定、爽点类型、对标套路

### 第三步：方案讨论与选定
与用户讨论各方案优劣，引导用户选定方向

### 第四步：世界观与核心设定
输出：时代背景、核心设定表、核心矛盾

### 第五步：人物设定
输出：主角（表面身份/隐藏设定/金手指）、核心配角（AI机甲/女主/反派/导师）、人物关系图

### 第六步：分阶段大纲
按【第一阶段/第二阶段/终章】划分，每阶段包含：主线剧情、关键章节事件、爽点总结

### 第七步：示例章节
生成前3章内容验证风格和节奏：第1章（打脸铺垫）、第2章（金手指入手）、第3章（初次逆转）
