## 工作流程

### 第一步：跨平台热点概览
- 调用 `get_hotspot_list` 工具（platform: "all"）或 `cross_platform_hotspot_search` 获取全平台热榜数据
- 按类别分组展示（科技/数码、游戏/电竞、二次元/娱乐、社会类）
- 标注每个热点的平台来源、核心价值和传播点
- 输出结构化表格，清晰呈现热点概览

### 第二步：热点筛选与创作价值识别
- 评估每个热点是否与用户创作方向匹配（AI、科技、机甲、二次元等）
- 标注「对你创作有价值的洞察」
- 标注与用户擅长领域契合的热点，提供简要创作关联提示
- 推荐优先深挖的方向

### 第三步：信息深挖
- 调用 `global_search` 或 `zhihu_global_search`/`zhihu_search` 搜索热点详情
- 获取硬参数、知乎讨论热度、高赞观点
- 整理核心事实：事件背景、数据指标、舆论焦点
- 输出为核心信息表格

### 第四步：创作切入点评估
- 输出「可以蹭热点的角度」表格，评估可行性
- 从多个维度提供创作切入思路：
  - **蹭热点文章**：时事评论、观点输出
  - **素材融合**：将热点元素融入现有小说世界观
  - **独立创作**：以热点为灵感开新坑
- 标注每个角度的可行性评分
- 推荐1-3个最优创作方向
- 结合用户现有项目特点给出建议

### 第五步：生成创作内容
- 根据选定方向，直接输出完整内容（文章/大纲/设定）
- **科技评论类**：硬参数→缺点分析→价值肯定→趋势预测→结语
- **小说类**：主角人设+世界观+大纲+示例章节

### 第六步：多方案对比推荐
- 提供多个方案供选择
- 每个方案标注「爽感」「差异化」「是否延续用户已有套路」
- 明确给出推荐建议和理由

### 第七步：行动建议
- 结合用户当前项目状态，推荐最适合的创作方式
- 提供具体的下一步行动选项

### 输出格式
- 热点概览：分类表格
- 核心信息：结构化摘要
- 创作分析：多角度表格 + 可行性评分
- 行动建议：具体推荐方向
