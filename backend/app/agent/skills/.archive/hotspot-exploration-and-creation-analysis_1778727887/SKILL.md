## 工作流程

### 第一步：获取跨平台热点概览
- 调用 `get_hotspot_list` 工具，设置 `platform: "all"`，获取多平台热榜数据
- 将热点按类别（科技/游戏/二次元/社会）分组整理
- 输出结构化表格，标注每个热点的平台来源

### 第二步：识别创作价值
- 从热点中筛选与创作相关的洞察
- 标注与用户擅长领域（如AI、二次元、科幻）契合的热点
- 提供简要的创作关联提示

### 第三步：深入热点详情（当用户选择特定热点时）
- 调用搜索工具（`zhihu_global_search` 或 `zhihu_search`）获取该热点深度信息
- 整理核心事实：事件背景、数据指标、舆论焦点

### 第四步：创作角度分析
- 从多个维度提供创作切入思路：
  - **蹭热点文章**：时事评论、观点输出
  - **素材融合**：将热点元素融入现有小说世界观
  - **独立创作**：以热点为灵感开新坑
- 标注每个角度的可行性评分

### 第五步：给出行动建议
- 结合用户当前项目状态，推荐最适合的创作方式
- 提供具体的下一步行动选项

### 输出格式
- 热点概览：分类表格
- 核心信息：结构化摘要
- 创作分析：多角度表格 + 可行性评分
- 行动建议：具体推荐方向
