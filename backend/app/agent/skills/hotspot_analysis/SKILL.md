# 知乎热点分析

## 工作流程

### 第一步：获取热榜
调用 `zhihu_hot_list` 工具获取知乎实时热榜数据。

### 第二步：分析热点
对获取到的热点进行分析：
1. **话题分类**：将热点按领域分类（科技、社会、娱乐、财经等）
2. **创作价值评估**：评估每个话题的创作潜力（讨论空间、受众广度、时效性）
3. **角度挖掘**：为高价值话题提供 2-3 个独特的切入角度

### 第三步：推荐选题
推荐 3-5 个最有价值的话题，格式：
- 话题标题 + 热度
- 为什么值得写
- 推荐角度（2-3 个）

### 第四步：深度调研（可选）
如果用户对某个话题感兴趣，调用 `zhihu_search` 搜索已有讨论，分析：
- 已有回答的主流观点
- 尚未被充分讨论的角度
- 可以差异化切入的方向

## 注意事项
- 热榜 API 每日限 100 次，已有缓存（1小时）
- 重点分析话题的「创作空间」而非简单转述标题
